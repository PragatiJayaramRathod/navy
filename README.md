## Deployment

This project was deployed on a cloud server using the following tools:

- Launched a virtual server using Amazon EC2 on Amazon Web Services.
- Connected to the EC2 instance using MobaXterm.
- Installed and configured Apache Tomcat on the server.
- Deployed the WAR file into the Tomcat `webapps` directory.
- Started the Tomcat server and successfully accessed the project through the browser.

This process demonstrates practical knowledge of cloud deployment, remote server management, and Java web application hosting.
